# koa2-graphql-mongodb ![](https://img.shields.io/badge/node-7.7.4-blue.svg)
A simple GraphQL server with Koa2 and MongoDB

## Installation
```
git clone https://github.com/nikitamarcius/koa2-graphql-mongodb

$ cd koa2-graphql-mongodb
$ npm i
$ npm run dev
```
